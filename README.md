Catálogo de Filmes

Projeto desenvolvido para fins acadêmicos, consistindo em um CRUD completo de filmes, com frontend em React e backend em Node.js, integrados por meio de uma API REST.

Tecnologias Utilizadas
Frontend
React
React Router DOM
Axios
CSS

Backend
Node.js
Express
SQLite (via sql.js – WebAssembly)
CORS

Deploy
Frontend: Netlify
Backend: Render

Links do Projeto
Frontend (Netlify):
(https://catalogo-filmeprojweb2.netlify.app/)

Backend (Render):
https://catalogo-filmes-qb8e.onrender.com

Funcionalidades
Listar filmes
Visualizar detalhes do filme
Cadastrar novo filme
Editar filme
Excluir filme
Exibir imagem (capa do filme)

Como Executar o Projeto Localmente
Backend
cd backend
npm install
node server.js

Acesse:
http://localhost:3001

Frontend
cd frontend
npm install
npm start

Acesse:
http://localhost:3000
